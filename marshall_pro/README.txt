HOW TO RUN
═══════════════════════════════════════════════

STEP 1 — Install libraries (only once)
───────────────────────────────────────
pip install flask numpy scipy matplotlib

STEP 2 — Run the app
───────────────────────────────────────
python app.py

STEP 3 — Browser opens automatically
───────────────────────────────────────
Go to: http://localhost:5000

STEP 4 — Fill your data
───────────────────────────────────────
- Enter both names (mandatory)
- Enter constants
- Fill sample observations
- Click GENERATE REPORT

STEP 5 — Print / Save PDF
───────────────────────────────────────
Click "Print / Save as PDF" button
Select "Save as PDF" in browser
